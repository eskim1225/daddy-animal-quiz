[Objective] 현재 화면 하단에 위치한 '컨트롤 패널(ControlPanel)'의 레이아웃 안정성을 확보하고, 동물 이미지 렌더링 방식을 개선해 줘.

[Context]

컨테이너의 padding, margin, 혹은 flex 속성을 세밀하게 조정해서 버튼이 화면 안에 온전히 보이도록 해줘.
3. 버튼 텍스트 줄바꿈 처리: 보조 버튼 중 '우리아이 정답 인정' 텍스트가 한 줄이라 답답해.

  3. 정렬 로직: 데이터가 뷰(View) 영역으로 넘어가기 전, 게임 로직 단계에서 각 동물의 `hints` 배열 내부를 `hintOrder` 값 기준의 오름차순으로 정렬(Sort)하여 항상 원하는 순서대로 렌더링되게 구현해야 해.

[Key Result]
1. 작업 전 백업 (Git Backup): 이번엔 안해도 괜찮아.
2. 계획 먼저 제시 (Plan First): 코드를 바로 수정하거나 작성하지 마. 위 요구사항을 해결하기 위해 어떤 파일들을 어떻게 고칠 것인지 단계별 '수정 계획(implementation_plan.md)'을 먼저 보고해 줘.
3. 승인 후 실행 (Execute on Approval): 내 승인(컨펌)이 떨어지면, 그때 기존 코드를 망가뜨리지 않도록 유의하며 전체 수정된 코드를 제공해 줘.
4. 코드 문서화: 수정된 코드에는 왜 `sort`나 `filter` 메서드를 사용했는지, 이 로직이 어떤 역할을 하는지 알 수 있도록 핵심 로직 부분에 반드시 '한국어 주석'을 상세히 달아줘.

   02) 프롬프트 02 - 요구사항에 맞춰 생성된 프롬프트
**[Role]**
당신은 5~7세 유아의 인지 발달과 눈높이에 완벽하게 맞춰 소통하는 '최고 수준의 유아 교육 전문가'이자, 서비스 기획 의도를 정확히 파악하여 정교한 JSON 데이터를 구축하는 '수석 데이터 엔지니어'입니다.

**[Objective]**
5~7세 아이들이 인지할 수 있는 친숙한 **동물 40~50종을 선별**하고, 각 동물마다 **단계별로 쉬워지는 10고개 힌트**와 저작권 문제가 없는 **Wikimedia Commons의 이미지 URL**을 결합하여 무결점의 **JSON 데이터**를 생성하는 것입니다.

**[Context]**
- **서비스 목적**: '아빠와 동물 10고개'라는 유아 대상 모바일 웹 퀴즈 서비스입니다. 부모(아빠)가 화면에 나오는 힌트를 보고 직접 읽어주거나 흉내 내며 아이와 상호작용하는 오프라인 유대감 형성이 핵심입니다.
- **힌트의 난이도 설계 (1~10단계)**:
  - 1번 힌트는 동물의 생태나 서식지 등 약간 추상적이고 어려운 내용으로 시작합니다.
  - 단계가 진행될수록 점점 직관적으로 변하며, 9번~10번 힌트는 특징적인 울음소리나 가장 대표적인 특징을 나타내는 아주 쉬운 힌트여야 합니다.
- **연기(Acting) 요소**: 한 동물의 10개 힌트 중 최소 2~3개는 부모가 행동이나 소리를 과장되게 흉내 내도록 유도하는 지문(예: `(아빠가 코를 길게 만들며 "뿌우~" 소리내주세요)`)을 텍스트 내에 포함해야 합니다.
- **이미지 확보**: 각 동물의 이미지는 `wikimedia.org` (Wikimedia Commons)에 호스팅된 저작권 프리(Public Domain 또는 CC) 고화질 이미지의 절대 경로(.jpg 또는 .png 등)로 추출해야 합니다.

**[Key Result]**
아래의 규칙과 포맷을 엄격히 준수하여 결과물을 도출하세요.

1. **동물 리스트**: 강아지, 고양이, 호랑이, 코끼리, 토끼 등 5~7세가 명확히 아는 동물 40~50종으로 구성.
2. **힌트 데이터**: 동물당 정확히 10개의 힌트(`hints` 배열). 아빠의 연기 가이드가 포함된 힌트의 경우 `acting: true`로 설정.
3. **이미지 데이터**: 반드시 실제 접속이 가능한 `https://upload.wikimedia.org/...` 형식의 이미지 URL 포함.
4. **출력 형식**: 오직 아래의 JSON 배열 형태로만 출력할 것 (불필요한 설명 생략). 데이터가 길 경우 여러 번에 나누어 출력해도 좋으니 구조를 깨뜨리지 말 것.

**[JSON 구조 예시]**
```json

  {
